医学信息沟通代表

Location 全国各地
Job Function 销售部
Position Summary
分析公司预算及实际财务状况，为各部门提供有价值的财务建议。
主要责任
1.区域产品的推广工作； 2.区域内专家的培养和维护； 3.销售市场活动的计划并执行； 4.及时反馈客户使用产品的相关信息。
任职资格
1.专科学历及以上，医药类专业优先； 2.1年以上相关医药产品销售工作经验者优先； 3.沟通技能； 4.分析和解决问题的能力； 5.计划组织的能力； 6.学习能力。

地区经理

Location 天津
Job Function 销售部
主要责任
1.根据公司要求，对负责的区域内目标医院进行合理的划分和管理； 2.依据公司的销售策略，对区域内目标医院制定销售策略和活动计划，领导并帮助代表完成上级下达的销售任务； 3.对区域内和各项相关政策熟悉并保持高度的敏感度，获取政策调整信息及关键人，参与影响区域内政策的制订，确保公司利益最大化； 4.负责招聘、培养、考核下属员工，通过协同拜访对下属员工进行反馈和辅导，不断提高其工作技能和绩效。 5.督导下属员工的日常工作，执行公司的推广策略和项目。 6.拜访区域内重要客户，确保其对公司及产品的认可度。 7.学术研讨会等活动的组织、策划、协调及跟进反馈工作。
任职资格
1.大专或以上学历，医药学类或管理类专业（含市场营销）优先； 2.3年以上医药行业销售经验或者1年以上销售管理岗位经验； 3.具备良好的问题处理能力和沟通协调能力，能承受较大工作压力； 4.具备良好的执行力，能够坚持原则，严格执行公司的各项管理制度； 5.能适应出差； 6.建立良好的团队凝聚力；

医学经理

Location 上海
Job Function 市场部
主要责任
1.对市场部提供医学支持，负责药物安全数据的管理； 2.制定医学产品计划并全面实施； 3.宣传资料的批准和合规； 4.“拜访助手”中产品内容整理。 5.提供医疗信息。 6.药物警戒-不良反应报告。 7.完成领导交代的其他任务。
任职资格
1.相关工作经验5年以上； 2.本科以上。 医学、药学专业优先； 3.良好的沟通能力； 4.良好的团队协作能力；

产品经理

Location 天津
Job Function 销售部
主要责任
1.基于产品分析和市场趋势评估，提供客户与竞品活动信息，协助制定产品整体市场策略及计划； 2.在所负责的区域或科室与销售力量进行跨部门合作，发现市场机会，为市场部提供相应报告，为调整市场策略和计划提供支持； 3.在市场活动执行过程中保持高度的专业化和有效性，确保推广方案的高效执行； 4.协助市场经理制作和更新产品推广资料，如产品信息、循证医学、治疗方案和产品宣传手册等。 5.与销售部门进行协作，为业务团队提供销售指南，进行推广工具的开发。 6.维护与更新专家库信息，并进行KOL活动管理。 7.定期为销售团队提供产品知识培训。
任职资格
1.大学本科以上学历； 2.专业要求临床医学、生物制药、化学、市场营销等相关专业； 3.工作经验5年以上制药行业工作经验，有销售经验为佳； 4.具备优秀的产品知识和学术背景； 5.丰富市场活动和推广经验； 6.具备很强的创新思维、学习能力，分析问题及独立解决问题能力强； 7.具备良好的沟通技巧、跨部门协作能力和执行力；

商务经理

Location 太原
Job Function 市场准入
主要责任
1.完成区域商业销售指标，并确保区域内商业合理库存； 2.与区域内关键商业客户建立战略合作伙伴关系，及时了解商业环境变化； 3.与区域内商业合作，充分利用商业资源，积极开展创新工作； 4.与区域内的招标主体机构人员建立良好沟通，了解招标操作流程和细节，配合相关同事解决招标期间存在的问题及突发、特殊事件等。 5.配合招标经理、大区商务总监与销售部充分沟通，选择区域内医院配送商。 6.严格遵守公司的合规程序、合规制度及财务制度等等，严格控制费用在预算之内。 7.完成领导交代的其他任务。
任职资格
1.大专及以上学历； 2.3年以上医药商业流通经验； 3.沟通以及谈判技巧； 4.熟练操作计算机及excel等软件； 5.诚信，良好的团队合作精神；
